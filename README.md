# Pet project: King tricks calculator

Tricks points and party score calculator for card game "King"

### Technologies used in the project:

* Android Jetpack Components
* Material Design
* Hilt
* Room
* Navigation
* viewBinding
* Coroutines/Flows
* Clean architecture, MVVM
